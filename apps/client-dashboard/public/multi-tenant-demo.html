üé® Key Features of Tenant-Specific Customization:

  • Isolated Configurations: Each tenant's widget settings are completely separate
  • Unique Storage: Configurations saved with tenant-specific storage keys
  • Tenant Context: URLs include tenant ID, timezone, and currency
  • Brand Consistency: Each tenant can match their restaurant's branding
  • Export/Import: Configurations can be backed up and transferred
  • Real-time Updates: Changes in widget management immediately affect that tenant's widgets
